# Voxel-Creator
Algorithm written to take in micro-ct scans, selects a 100x100px region and converts to a voxel model .
The scans used are from https://bone3d.zib.de/data/2005/ESA29-99-L3/
Using the images as is, will result in a voxel model with a lot of noise, it is recommended to remove the noise
from the images first. This is not done in this algorithm
